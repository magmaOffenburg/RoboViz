General
F1 Display this help page
F2 Open configuration window
F4 Switch between log mode and live mode
Alt + F4 Close RoboViz
Ctrl + F Toggle fullscreen
Alt + Enter Toggle fullscreen
F11 Toggle fullscreen
F12 Take a screenshot (saved in /screenshots)
Escape Close dialogs
F Toggle 2D field overlay
Q Toggle list of fouls
I Change annotation mode (player numbers, drawings, none)
N Toggle number of active players per team
Y Display drawings panel
T Toggle all drawings

Selection
Left mouse click (De)select the ball / a player
0 Select ball
Ctrl + [F1-F11] Select player (left team)
Ctrl + Shift + [F1-F11] Select player (right team)
Tab Select next player in team
Shift + Tab Select previous player in team
Escape Cancel current selection
Ctrl + left mouse click Beam selected object to cursor position (live mode)
Ctrl + Alt + left mouse click Move ball to selected position (live mode)
Shift + Alt + left mouse click Launch ball to selected position at a 45 degree angle (live mode)
V Toggle first person perspective (on selected player)
E Toggle third person perspective (on selected player)

Robot Vantage
W / Up / Mouse wheel up Decrease FOV
S / Down / Mouse wheel down Increase FOV

Camera
Drag left mouse Rotate camera
Hold right mouse Move camera up
Page up Move camera up
Shift + hold right mouse Move camera down
Page down Move camera down
Mouse wheel up Move camera forward
Mouse wheel down Move camera backward
1 - 8 Sets camera to predefined positions
Space Toggle ball tracking
Shift + Space Toggle player tracking
WASD Directional camera movement
Arrow keys Directional camera movement
Hold Shift Precise camera movement

Live Mode
O Display playmode overlay
L Free kick left
R Free kick right
Shift + L Direct free kick left
Shift + R Direct free kick right
B Drop ball
K Kick off left
J Kick off right
C Connect to server (happens automatically by default)
Ctrl + C Open the "Connect to..." dialog
U Request full state update from server
M Toggle showing server real time speed percentage
Shift + X Kill the server
Shift + T Reset the time

Playmode Overlay
Letters, _ and backspace Edit the filter text
Down arrow Select next playmode
Up Arrow Select previous playmode
Enter Set playmode to selection

Logfile Mode
P Toggle pause / play
R Rewind to start of log file
X Decrease playback speed
C Increase playback speed
Comma Step backward (while paused)
Period Step forward (while paused)
G Jump to previous goal
H Jump to next goal
